Fundamentos de Banco de Dados

SGBD | Modelagem | SQL | Aplicações

Os bancos de dados são a espinha dorsal dos sistemas de informação modernos, armazenando e gerenciando dados de forma estruturada e eficiente.

Eles permitem que aplicações capturem, armazenem, recuperem e analisem informações, transformando dados brutos em conhecimento valioso para tomada de decisões.

O que é um SGBD? Tipos e Aplicações

Definição | Classificação | Utilização

O SGBD (Sistema Gerenciador de Banco de Dados) é um software responsável por armazenar, organizar e manipular dados de forma estruturada. Ele garante segurança, consistência e acesso controlado às informações.

Dicionário de Dados e Tipos

Definição | Estrutura | Tipos de Dados

O dicionário de dados documenta todas as informações de um banco de dados: nomes de tabelas, colunas, tipos e restrições. É essencial para manter a padronização e clareza do sistema.

Exemplo de Dicionário de Dados - Tabela "Clientes"

Campo Tipo Tamanho Descrição Restrições
id_cliente INT 11 Identificador único do cliente PRIMARY KEY, AUTO_INCREMENT
nome VARCHAR 100 Nome completo do cliente NOT NULL
email VARCHAR 255 E-mail do cliente UNIQUE, NOT NULL
data_nascimento DATE - Data de nascimento NULL
ativo BOOLEAN - Status do cliente DEFAULT TRUE

Tipos de Dados Principais em SQL

Numéricos

  • INT - Números inteiros
  • DECIMAL - Números decimais precisos
  • FLOAT - Números de ponto flutuante
  • BOOLEAN - Valores TRUE/FALSE

Textuais

  • VARCHAR(n) - Texto de tamanho variável
  • CHAR(n) - Texto de tamanho fixo
  • TEXT - Textos longos
  • ENUM - Lista de valores pré-definidos

Data/Hora

  • DATE - Data (YYYY-MM-DD)
  • DATETIME - Data e hora
  • TIMESTAMP - Marca temporal
  • TIME - Horário (HH:MM:SS)

Modelagem de Banco de Dados | DER e MER

Conceito | Estrutura | Diferença

A modelagem de banco de dados é o processo de planejar como os dados serão organizados e relacionados. Ela garante integridade, consistência e fácil manutenção.

DER - Diagrama Entidade-Relacionamento

Representação gráfica das entidades e seus relacionamentos

▢ Entidades - Retângulos
○ Atributos - Elipses
◇ Relacionamentos - Losangos

MER - Modelo Entidade-Relacionamento

Versão extendida do DER com mais detalhes conceituais

1:1 Um para Um
1:N Um para Muitos
N:M Muitos para Muitos
