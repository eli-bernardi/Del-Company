Sensores IoT

Catálogo técnico completo de sensores para Internet das Coisas e Automação Industrial

1. O Conceito de Sensoriamento

O sensoriamento é o processo de detecção, medição e conversão de grandezas físicas do mundo real (como temperatura, pressão, luminosidade, presença ou fluxo) em sinais elétricos que podem ser interpretados, processados e armazenados por sistemas eletrônicos. No contexto da tecnologia, os sensores atuam como os "olhos e ouvidos" de qualquer sistema inteligente, traduzindo o estado analógico do ambiente para a linguagem digital dos microcontroladores e computadores.

2. A Importância dos Sensores na Automação Industrial (Indústria 4.0)

Na era da Indústria 4.0, os sensores são a base para a digitalização das fábricas. Eles permitem:

  • 📊 Monitoramento em Tempo Real — Acompanhamento instantâneo de variáveis críticas em linhas de produção para evitar falhas de processos.
  • 🔧 Manutenção Preditiva — Identificação de padrões anômalos (como vibrações excessivas ou superaquecimento) antes que uma máquina quebre.
  • ⚡ Eficiência Energética e de Recursos — Controle preciso do uso de insumos, água, gases e energia elétrica com base na demanda real aferida.
  • 🛡️ Segurança Operacional — Bloqueio automático de maquinário industrial quando operadores entram em áreas de risco.

3. A Relação entre Sensores, Comunicação de Dados e Controle de Processos (IoT)

A arquitetura clássica de um sistema de Internet das Coisas (IoT) e Automação Industrial baseia-se em um ciclo fechado de três etapas:

1

Aquisição (Sensoriamento)

Os sensores capturam as variáveis físicas e as transformam em sinais elétricos.

2

Transmissão (Comunicação de Dados)

Os dados coletados são enviados através de protocolos de comunicação (I2C, SPI, UART, Wi-Fi, LoRaWAN, MQTT) para controladores ou nuvem.

3

Ação (Controle de Processos)

Microcontroladores (como ESP32) ou CLPs processam os dados e tomam decisões automáticas.

4. O Ecossistema Arduino

  • Conceito: O Arduino é uma plataforma de prototipagem eletrônica de código aberto (open-source), baseada em hardware e software flexíveis e fáceis de usar.
  • Funcionamento: A placa Arduino possui um microcontrolador (ex: ATmega328P no Arduino Uno) rodeado por pinos de entrada e saída (I/O). Ela lê as entradas (sinal digital/analógico) e escreve nas saídas (LEDs, motores, transmissão de dados).
  • Programação: Os programas do Arduino são chamados de sketches e escritos em C/C++. A estrutura básica do código divide-se em duas funções principais: 
    void setup() {
  // Código executado uma única vez ao iniciar a placa
}
void loop() {
  // Código executado repetidamente em ciclo contínuo
}

1. DHT11

🌡️ Temperatura e Umidade

Conceito: Sensor básico integrado para medição simultânea de temperatura ambiente e umidade relativa do ar.

Princípio de Funcionamento: Utiliza um componente resistivo para medição de umidade (NTC/polímero) e um termistor para temperatura. Um microcontrolador interno processa os dados físicos e envia os valores por transmissão de fio único (Single-Bus).

Especificações Técnicas

  • Tensão de Operação: 3V a 5.5V CC
  • Faixa de Medição de Umidade: 20% a 90% UR (Precisão: ±5%)
  • Faixa de Medição de Temperatura: 0°C a 50°C (Precisão: ±2°C)
  • Tempo de resposta: 2 segundos

Tipo de Sinal: Digital (Protocolo proprietário de 1 fio - Single-Wire)

Aplicações Industriais / IoT: Sistemas residenciais de climatização (HVAC), estações meteorológicas domésticas, controle de umidade em salas de servidores.

Exemplo de Utilização: Monitoramento ambiental de salas comerciais para acionamento automático de umidificadores de ar quando a umidade cai abaixo de 40%.

Fabricantes / Modelos Comerciais: ASAIR, Keyestudio, DF Robot.

2. DHT22 (AM2302)

🌡️ Temperatura e Umidade

Conceito: Sensor capacitivo de alta precisão para medição de umidade relativa e temperatura ambiente.

Princípio de Funcionamento: Semelhante ao DHT11, porém utiliza um sensor de umidade capacitivo de alta qualidade e um termistor de maior precisão, gerando leituras mais estáveis e abrangentes transmitidas via sinal digital de barramento único.

Especificações Técnicas

  • Tensão de Operação: 3.3V a 6V CC
  • Faixa de Medição de Umidade: 0% a 100% UR (Precisão: ±2%)
  • Faixa de Medição de Temperatura: -40°C a 80°C (Precisão: ±0.5°C)
  • Tempo de resposta: 2 segundos

Tipo de Sinal: Digital (Protocolo de 1 fio)

Aplicações Industriais / IoT: Estufas agrícolas de precisão, galpões de armazenamento de alimentos perecíveis, laboratórios farmacêuticos.

Exemplo de Utilização: Controle climático em incubadoras avícolas, mantendo a temperatura em exatos 37.5°C com variação máxima de 0.5°C.

Fabricantes / Modelos Comerciais: ASAIR, Aosong Electronics.

3. LM35

🌡️ Temperatura

Conceito: Sensor de temperatura de precisão com saída analógica linear de tensão calibrada diretamente em graus Celsius.

Princípio de Funcionamento: Baseia-se na variação física da queda de tensão de uma junção de diodo semicondutor de silício conforme a temperatura altera. Apresenta uma saída linear de 10mV por cada 1°C de variação.

Especificações Técnicas

  • Tensão de Operação: 4V a 30V CC
  • Faixa de Medição: -55°C a +150°C
  • Precisão garantida: ±0.5°C (em temperatura ambiente de 25°C)
  • Fator de escala linear: +10 mV/°C

Tipo de Sinal: Analógico

Aplicações Industriais / IoT: Proteção contra superaquecimento em fontes de alimentação, monitoramento de dissipadores de calor e painéis elétricos.

Exemplo de Utilização: Medição de temperatura de dissipadores de inversores de frequência industriais, desligando o circuito preventivamente caso a leitura ultrapasse 85°C.

Fabricantes / Modelos Comerciais: Texas Instruments, Analog Devices.

4. DS18B20

🌡️ Temperatura

Conceito: Termômetro digital inteligente de resolução configurável, altamente recomendado para medição de temperaturas em ambientes hostis e líquidos por meio de sua versão blindada à prova d'água.

Princípio de Funcionamento: Converte a temperatura física interna em uma palavra digital de até 12 bits. Comunica-se usando o protocolo de barramento 1-Wire, permitindo ligar múltiplos sensores no mesmo pino com identificadores únicos.

Especificações Técnicas

  • Tensão de Operação: 3.0V a 5.5V CC
  • Faixa de Medição: -55°C a +125°C (Precisão de ±0.5°C na faixa de -10°C a +85°C)
  • Resolução selecionável: de 9 a 12 bits
  • Interface de comunicação: 1-Wire

Tipo de Sinal: Digital (Protocolo 1-Wire)

Aplicações Industriais / IoT: Sistemas de refrigeração industrial, tanques de fermentação de cerveja, controle de temperatura de óleos hidráulicos.

Exemplo de Utilização: Sistema de controle de temperatura de uma caldeira de água encamisada, monitorando o líquido constantemente sem risco de curto-circuito devido ao encapsulamento de aço inoxidável à prova d'água.

Fabricantes / Modelos Comerciais: Maxim Integrated / Analog Devices.

5. LDR (Fotoresistor)

💡 Luminosidade

Conceito: Sensor fotoelétrico analógico cuja resistência elétrica varia de forma inversamente proporcional à intensidade da luz incidente sobre a sua superfície de sulfeto de cádmio (CdS).

Princípio de Funcionamento: Quando a luz incide no material semicondutor do LDR, há liberação de elétrons livres, aumentando a condutividade elétrica e reduzindo a resistência. No escuro, a resistência sobe para a faixa de megaohms; sob luz forte, cai para algumas dezenas de ohms.

Especificações Técnicas

  • Resistência no Escuro: Tipicamente > 1 MΩ
  • Resistência na Luz (10 lux): 8 kΩ a 20 kΩ
  • Tensão máxima suportada: 150V CC
  • Pico de sensibilidade espectral: 540 nm (próximo ao olho humano)

Tipo de Sinal: Analógico (Necessita de circuito divisor de tensão resistivo para leitura)

Aplicações Industriais / IoT: Sistemas de iluminação pública inteligente, fotômetros industriais simples, detectores de chamas.

Exemplo de Utilização: Poste de iluminação pública inteligente que monitora a luz do dia e aciona uma lâmpada LED quando a luz natural cai abaixo de um limiar crítico.

Fabricantes / Modelos Comerciais: Adafruit, SparkFun, marcas genéricas.

6. BH1750 (BH1750FVI)

💡 Luminosidade

Conceito: Sensor digital de luz ambiente (luxímetro) de alta precisão com calibração interna direta na unidade padrão "lux".

Princípio de Funcionamento: Integra um fotodiodo de silício acoplado a um conversor analógico-digital (ADC) interno de 16 bits. Capta a luz, realiza a conversão digital precisa e disponibiliza os dados diretamente via protocolo I2C.

Especificações Técnicas

  • Tensão de Operação: 3V a 5V CC
  • Faixa de Medição: 1 a 65.535 lux
  • Resolução: Até 0.5 lux (modo de alta resolução)
  • Influência de luz infravermelha: Mínima

Tipo de Sinal: Digital (Protocolo I2C)

Aplicações Industriais / IoT: Ajuste automático de brilho em displays industriais, controle avançado de iluminação de estufas hidropônicas e automação predial sustentável.

Exemplo de Utilização: Integração em sistema IoT agrícola para regular a intensidade de painéis de luz LED artificiais dedicados ao crescimento de plantas.

Fabricantes / Modelos Comerciais: ROHM Semiconductor.

7. HC-SR04

📏 Distância

Conceito: Sensor de proximidade ultrassônico capaz de medir distâncias de objetos sem contato físico através de ondas sonoras de alta frequência.

Princípio de Funcionamento: Emite um pulso ultrassônico (40 kHz) pelo transmissor (Trigger). A onda viaja pelo ar, bate no obstáculo e retorna, sendo captada pelo receptor (Echo). O microcontrolador calcula a distância medindo o tempo decorrido entre o envio e o recebimento do sinal.

Especificações Técnicas

  • Tensão de Operação: 5V CC
  • Corrente de Trabalho: 15 mA
  • Faixa de Medição: 2 cm a 400 cm (Precisão: ±3 mm)
  • Ângulo de medição: < 15°

Tipo de Sinal: Digital (Pulsos de tempo nos pinos Trigger e Echo)

Aplicações Industriais / IoT: Robótica industrial móvel (AGVs), medição de nível em silos, sistemas de auxílio de estacionamento.

Exemplo de Utilização: Sensor montado no topo de uma lixeira urbana inteligente para monitorar o preenchimento volumétrico, disparando alerta quando a distância for menor que 10 cm.

Fabricantes / Modelos Comerciais: ElecFreaks, SparkFun, DF Robot.

8. PIR HC-SR501

🚶 Movimento e Presença

Conceito: Sensor infravermelho passivo (PIR) projetado para detectar o movimento de corpos quentes (seres humanos e animais) em sua área de cobertura.

Princípio de Funcionamento: Detecta a radiação infravermelha naturalmente emitida pelos corpos vivos. Uma lente de Fresnel foca os raios de calor na célula piroelétrica interna. Ao notar uma variação repentina de temperatura, o sensor envia um sinal de nível lógico alto (HIGH).

Especificações Técnicas

  • Tensão de Operação: 4.5V a 20V CC
  • Tensão de Saída: Digital 3.3V (HIGH) / 0V (LOW)
  • Tempo de retardo ajustável: 0.5s a 200s
  • Faixa de Detecção: até 7 metros (ângulo < 100°)

Tipo de Sinal: Digital

Aplicações Industriais / IoT: Sistemas de alarmes e segurança patrimonial, automação de iluminação para redução de consumo, controle de acesso a salas de servidores.

Exemplo de Utilização: Sistema de segurança de um galpão industrial que monitora movimentações fora do expediente, ativando câmera IP e disparando alertas sonoros.

Fabricantes / Modelos Comerciais: Adafruit, Keyestudio.

9. Sensor Indutivo LJ12A3-4-Z/BX

🔩 Proximidade (Industrial)

Conceito: Sensor de presença industrial sem contato físico utilizado exclusivamente para detectar a proximidade de objetos ferrosos e metálicos.

Princípio de Funcionamento: Gera um campo eletromagnético alternado de alta frequência em sua face ativa. Quando um objeto metálico entra nesse campo, correntes parasitas (Correntes de Foucault) são induzidas, reduzindo a amplitude das oscilações e acionando o chaveamento eletrônico.

Especificações Técnicas

  • Tensão de Alimentação: 6V a 36V CC
  • Distância de Detecção: 4 mm (para ferro/aço comum)
  • Tipo de Transistor: NPN NA (Normalmente Aberto)
  • Corrente máxima de saída: 300 mA

Tipo de Sinal: Digital

Aplicações Industriais / IoT: Contagem de peças metálicas em esteiras, sensores de fim de curso em fresadoras CNC, alinhamento de braços robóticos.

Exemplo de Utilização: Acoplamento em uma engrenagem de aço para atuar como sensor de rotação, contando as passagens dos dentes para calcular a velocidade de rotação.

Fabricantes / Modelos Comerciais: Pepperl+Fuchs, Autonics, Sick.

10. Sensor Capacitivo LJ18A3-8-Z/BX

🔩 Proximidade (Industrial)

Conceito: Sensor de presença sem contato físico de nível industrial capaz de detectar tanto metais quanto materiais não-metálicos (sólidos e líquidos).

Princípio de Funcionamento: Mede a variação da capacitância gerada entre a face sensora e o objeto. A presença do material altera a constante dielétrica, provocando uma variação capacitiva que ativa a saída.

Especificações Técnicas

  • Tensão de Alimentação: 6V a 36V CC
  • Distância de Detecção: 8 mm (ajustável por potenciômetro)
  • Tipo de Saída: NPN NA (Normalmente Aberto)
  • Corrente máxima de chaveamento: 300 mA

Tipo de Sinal: Digital

Aplicações Industriais / IoT: Detecção de nível de líquidos através de paredes de tanques plásticos, identificação de embalagens de papelão vazias, contagem de grãos.

Exemplo de Utilização: Sensor montado externamente em um tubo de plástico PVC para monitorar o nível alto de água sanitária sem contato corrosivo direto.

Fabricantes / Modelos Comerciais: Pepperl+Fuchs, Carlo Gavazzi, Autonics.

11. MQ-2

💨 Gás

Conceito: Sensor de detecção de gases inflamáveis de amplo espectro e de fumaça, ideal para prevenção de acidentes e segurança residencial/industrial.

Princípio de Funcionamento: Utiliza um elemento sensor de dióxido de estanho (SnO₂) que apresenta baixa condutividade em ar limpo. Quando há presença de gases combustíveis, a condutividade aumenta. Uma resistência de aquecimento interna acelera a reação química.

Especificações Técnicas

  • Tensão de Operação: 5V CC (Exige cerca de 150 mA)
  • Gases Detectáveis: GLP, Metano, Propano, Hidrogênio, Álcool e Fumaça
  • Faixa de Concentração: 300 a 10.000 ppm

Tipo de Sinal: Analógico e Digital (com comparador LM393 ajustável)

Aplicações Industriais / IoT: Alarmes industriais de vazamento de gás, sistemas de prevenção de incêndio, monitoramento ambiental de oficinas.

Exemplo de Utilização: Detector de fumaça inteligente que envia dados via IoT para cortar a válvula solenoide principal de combustível.

Fabricantes / Modelos Comerciais: Hanwei Electronics.

💨 Gás e Qualidade do Ar

Conceito: Sensor de gás de alta sensibilidade para monitoramento contínuo da qualidade do ar e detecção de gases poluentes e nocivos à saúde.

Princípio de Funcionamento: Baseia-se na variação da condutividade elétrica de um elemento sensor semicondutor metálico de aquecimento ativo (SnO₂) na presença de compostos orgânicos voláteis (COVs) e gases asfixiantes.

Especificações Técnicas

  • Tensão de Operação: 5V CC
  • Gases Detectáveis: Amônia (NH₃), Óxidos de Nitrogênio (NOx), Álcool, Benzeno, Fumaça e CO₂
  • Faixa de Concentração: 10 a 10.000 ppm

Tipo de Sinal: Analógico e Digital (com potenciômetro de limiar)

Aplicações Industriais / IoT: Estações urbanas de monitoramento ambiental, sistemas de purificadores de ar, ventilação mecânica automatizada.

Exemplo de Utilização: Dispositivo IoT de qualidade de ar em salas de aula que monitora CO₂ e aciona exaustor quando a saturação ultrapassa 1000 ppm.

Fabricantes / Modelos Comerciais: Hanwei Electronics.

Tipo de Sinal: Analógico

Aplicações Industriais / IoT: Irrigação inteligente em agricultura de precisão, monitoramento de solos em floriculturas, sistemas domésticos de rega.

14. Sensor de Chuva FC-37

🌧️ Chuva e Climatologia

Conceito: Sensor resistivo em forma de placa projetado para identificar a presença de gotas de água ou acúmulo de líquidos em sua face plana.

Princípio de Funcionamento: A placa possui trilhas de cobre niquelado dispostas em ziguezague sem contato elétrico. Ao cair uma gota de água, ela cria uma ponte resistiva fechando o circuito, reduzindo a resistência equivalente.

Especificações Técnicas

  • Tensão de Operação: 3.3V a 5V CC
  • Acabamento das Trilhas: Banho de níquel contra oxidação
  • Controle: Placa comparadora LM393 com potenciômetro de sensibilidade

Tipo de Sinal: Analógico e Digital (Chaveamento Liga/Desliga)

Aplicações Industriais / IoT: Fechamento automático de claraboias, recolhimento de toldos, acionamento de limpadores de para-brisas.

Exemplo de Utilização: Automação IoT de uma estufa de secagem de grãos que fecha automaticamente as janelas do teto ao detectar as primeiras gotas d'água.

Fabricantes / Modelos Comerciais: SparkFun, marcas genéricas.

15. Sensor de Nível (Boia Magnética)

💧 Nível

Conceito: Sensor do tipo interruptor on/off mecânico acionado por atração magnética, projetado para controle de reservatórios e caixas d'água.

Princípio de Funcionamento: Possui uma ampola de vidro com um contato elétrico sensível a campos magnéticos (Reed Switch). Um flutuador com ímã permanente sobe com o nível da água, alinhando-se com o Reed Switch e fechando/abrindo o circuito.

Especificações Técnicas

  • Tensão Máxima: até 220V CA/CC
  • Potência de Contato: 10W a 50W máximo
  • Pressão de Operação: < 1.0 MPa

Tipo de Sinal: Digital (Contato Seco NA/NF)

Aplicações Industriais / IoT: Prevenção de funcionamento a seco em bombas hidráulicas, alarmes de transbordamento, controle de caixas d'água.

Exemplo de Utilização: Chave boia no fundo de um poço artesiano para desligar a bomba de sucção caso o poço fique temporariamente vazio.

Fabricantes / Modelos Comerciais: Finder, JNG, Metaltex.

16. Sensor de Fluxo de Água YF-S201

💧 Vazão e Fluxo de Líquidos

Conceito: Sensor de medição volumétrica em linha e em tempo real para fluxo contínuo de líquidos.

Princípio de Funcionamento: O fluxo interno de líquido movimenta uma hélice com um ímã integrado. Um sensor de efeito Hall conta as voltas da hélice gerando pulsos elétricos de frequência correspondente à vazão.

Especificações Técnicas

  • Tensão de Operação: 5V a 18V CC (Corrente < 15 mA)
  • Faixa de Medição: 1 a 30 Litros/minuto
  • Frequência de Pulso: F = 7.5 × Q (Q em L/min)
  • Pressão máxima: ≤ 1.75 MPa (17.5 bar)

Tipo de Sinal: Digital (Trem de pulsos gerado por interrupção Hall)

Aplicações Industriais / IoT: Medição individual de consumo de água (telemetria IoT), dosagem automatizada de precisão, detecção de vazamentos.

Exemplo de Utilização: Sistema dosador que interrompe a passagem de água após detectar a contagem precisa de exatos 50 litros de vazão.

📟 Código 1: DHT22 (Temperatura e Umidade)

Utiliza a biblioteca Adafruit DHT para ler dados do sensor e exibi-los no Monitor Serial.

/*
 * Exemplo de Programação - Sensor DHT22
 * Unidade Curricular: Internet das Coisas - SENAI SC
 * Desenvolvido para o catálogo técnico digital
 */

#include "DHT.h"  // Inclui a biblioteca oficial para sensores DHT

#define DHTPIN 2     // Define que o pino digital do sinal do sensor é o D2
#define DHTTYPE DHT22 // Especifica que o sensor físico conectado é o DHT22

DHT dht(DHTPIN, DHTTYPE); // Cria o objeto dht configurando o pino e modelo

void setup() {
  Serial.begin(9600); // Inicializa a comunicação Serial a 9600 bps
  Serial.println("Inicializando leitura do DHT22...");
  dht.begin(); // Inicializa a comunicação digital interna com o sensor
}

void loop() {
  delay(2000); // Aguarda o intervalo padrão de 2 segundos entre leituras

  // Lê a umidade relativa (%) e a temperatura (°C) do sensor
  float umidade = dht.readHumidity();
  float temperatura = dht.readTemperature();

  // Verifica se a leitura foi bem-sucedida ou se houve falha de cabo
  if (isnan(umidade) || isnan(temperatura)) {
    Serial.println("Falha na leitura física do sensor DHT22!");
    return; // Retorna para o início do loop sem processar dados incorretos
  }

  // Imprime os valores lidos com precisão de formatação na porta serial
  Serial.print("Umidade do ar: ");
  Serial.print(umidade);
  Serial.print(" % | ");
  Serial.print("Temperatura: ");
  Serial.print(temperatura);
  Serial.println(" *C");
}

📟 Código 2: LM35 (Temperatura Analógica)

Leitura analógica direta e conversão matemática da tensão em graus Celsius.

/*
 * Exemplo de Programação - Sensor de Temperatura Analógico LM35
 */

#define LM35_PIN A0 // Conecta a perna de sinal analógico do LM35 no pino analógico A0

void setup() {
  Serial.begin(9600); // Inicializa a comunicação serial rápida
}

void loop() {
  // Realiza a leitura da conversão analógica padrão (retorna entre 0 e 1023)
  int leituraADC = analogRead(LM35_PIN);

  // Converte o valor do ADC de volta para milivolts (tensão elétrica real na porta)
  // O Arduino Uno trabalha com referência estável interna padrão de 5V (5000 mV)
  float tensao_mV = (leituraADC * 5000.0) / 1023.0;

  // Como o LM35 varia exatamente 10mV para cada 1°C de temperatura medida:
  float temperatura_C = tensao_mV / 10.0;

  // Imprime a informação de forma limpa na tela do Monitor Serial do Usuário
  Serial.print("Leitura Temperatura Real do LM35: ");
  Serial.print(temperatura_C);
  Serial.println(" *C");

  delay(1000); // Aguarda 1 segundo para a próxima amostragem estável
}

📟 Código 4: HC-SR04 (Ultrassônico de Distância)

Mede o tempo de eco do sinal ultrassônico e calcula a distância em centímetros. 

/*
 * Exemplo de Programação - Medição com Sensor de Distância Ultrassônico HC-SR04
 */

#define TRIG_PIN 9 // Pino digital D9 envia o pulso sonoro inicial (Trigger)
#define ECHO_PIN 8 // Pino digital D8 recebe o pulso refletido (Echo)

void setup() {
  Serial.begin(9600); // Inicializa o monitoramento serial
  pinMode(TRIG_PIN, OUTPUT);  // Pino TRIG configurado para emitir sinal
  pinMode(ECHO_PIN, INPUT);  // Pino ECHO configurado para ler sinal de retorno
}

void loop() {
  // Garante um pulso limpo em nível lógico baixo (LOW) inicial de segurança
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);

  // Envia um pulso em nível lógico alto (HIGH) de exatos 10 microssegundos para disparar
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);

  // Mede o tempo em microssegundos que o sinal levou para atingir a barreira e retornar
  long duracao_pulso = pulseIn(ECHO_PIN, HIGH);

  // Calcula matematicamente a distância em centímetros com base na velocidade do som
  float distancia_cm = (duracao_pulso * 0.0343) / 2.0;

  // Imprime os dados estruturados na saída Serial
  Serial.print("Distancia aferida do objeto: ");
  Serial.print(distancia_cm);
  Serial.println(" cm");

  delay(600); // Aguarda 600 milissegundos para realizar a leitura seguinte com segurança
}

📟 Código 6: Sensor Capacitivo de Umidade do Solo

/*
 * Exemplo de Programação - Sensor de Umidade do Solo Capacitivo
 */

#define SOLO_PIN A2 // Pino analógico A2 conectado à saída analógica do sensor de solo

// Definição dos valores padrão calibrados empiricamente em laboratório
const int valor_seco = 600;    // Valor quando o sensor está totalmente no ar seco
const int valor_molhado = 250; // Valor quando o sensor está inserido em terra úmida

void setup() {
  Serial.begin(9600); // Inicialização da comunicação com o monitor serial
}

void loop() {
  int leitura_crua = analogRead(SOLO_PIN); // Lê o valor elétrico (0 a 1023)

  // Mapeia matematicamente a leitura crua analógica em um valor percentual de umidade
  // Nota-se a inversão matemática na escala, já que menor tensão = maior teor de água
  int umidade_porcento = map(leitura_crua, valor_seco, valor_molhado, 0, 100);

  umidade_porcento = constrain(umidade_porcento, 0, 100);

  Serial.print("Leitura Analoga Crua: ");
  Serial.print(leitura_crua);
  Serial.print(" | Porcentagem de Umidade Estimada: ");
  Serial.print(umidade_porcento);
  Serial.println("%");

  if (umidade_porcento < 30) {
    Serial.println("Alerta: Solo muito seco! Necessita de irrigacao imediata.");
  } else {
    Serial.println("Solo devidamente hidratado.");
  }

  delay(2000); // Aguarda 2 segundos para manter uma frequência de irrigação aceitável
}

📟 Código 9: Sensor Hall A3144 (Velocidade e Rotação)

Utiliza interrupção externa para contar revoluções com máxima precisão de velocidade.

/*
 * Exemplo de Programação - Leitura de Rotação com Sensor Hall A3144
 */

#define HALL_PIN 2 // Pino digital D2 (interrupção INT0) conectado ao pino de saída do Hall
#define LED_PIN 13 // LED de indicação de pulso

volatile int contador_pulsos = 0; // Variável volátil para uso em interrupção
unsigned long tempo_anterior = 0;
float rpm = 0;

void setup() {
  Serial.begin(9600);
  pinMode(HALL_PIN, INPUT_PULLUP);
  pinMode(LED_PIN, OUTPUT);
  attachInterrupt(digitalPinToInterrupt(HALL_PIN), contarPulso, FALLING);
}

void loop() {
  unsigned long tempo_atual = millis();

  if (tempo_atual - tempo_anterior >= 1000) {
    // Calcula RPM (rotações por minuto) baseado na contagem de pulsos
    rpm = (contador_pulsos / 20.0) * 60.0;
    contador_pulsos = 0;
    tempo_anterior = tempo_atual;

    Serial.print("Velocidade: ");
    Serial.print(rpm);
    Serial.println(" RPM");
  }
}

void contarPulso() {
  contador_pulsos++;
  digitalWrite(LED_PIN, !digitalRead(LED_PIN));
}

/*

 */

#define VIBRACAO_PIN 5 // Pino digital D5 conectado na saída DO do módulo SW-420
