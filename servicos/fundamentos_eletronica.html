Fundamentos de Eletrônica

História | Segmentos | Utilidades | Aplicações

A eletrônica está presente em praticamente tudo que usamos no dia a dia, desde o celular que carregamos no bolso até os sistemas de iluminação, computadores, televisores e até carros. Os fundamentos de eletrônica são a base para entender como funcionam os circuitos, componentes e dispositivos que tornam possível toda essa tecnologia.

Esse campo do conhecimento envolve conceitos como corrente elétrica, tensão, resistência, capacitores, transistores e muito mais, servindo de alicerce para estudantes, técnicos e profissionais que desejam compreender e criar soluções tecnológicas.

Contexto Histórico

A história da eletrônica começa ligada ao estudo da eletricidade no século XVIII. Pesquisadores como Benjamin Franklin e Alessandro Volta deram os primeiros passos ao descobrir as propriedades da eletricidade e criar dispositivos como a pilha elétrica, que permitiu produzir corrente de forma contínua.

No século XIX, avanços como a Lei de Ohm (1827) e a invenção do telégrafo abriram caminho para aplicações práticas. Mas foi no início do século XX que a eletrônica realmente nasceu, com a invenção da válvula termiônica.

A partir da década de 1940, a invenção do transistor revolucionou tudo. Pequeno, barato e eficiente, ele substituiu as válvulas e tornou possível a criação dos computadores modernos.

Os anos 80 e 90 testemunharam a popularização dos computadores pessoais e dispositivos móveis, enquanto a revolução digital transformava a sociedade. Em 2005, surge o Arduino, que democratizou o acesso à eletrônica.

Hoje, vivemos em um mundo completamente dependente da eletrônica, presente em praticamente todos os dispositivos modernos.

Arduíno

Exemplificação | Funcionalidade | Utilização

O Arduino é uma plataforma de prototipagem eletrônica projetada para facilitar o desenvolvimento de projetos interativos. Ele combina hardware e software simples e intuitivos.

Criado em 2005, tornou-se popular por permitir o controle de sensores, motores, LEDs e outros componentes.

O funcionamento básico envolve escrever um código em C/C++ e enviá-lo via USB para o microcontrolador.

Exemplo prático: acender um LED quando um botão é pressionado.


Exemplo de código:


const express = require('express')
const app = express()
const { SerialPort, ReadLineParser } = require('serialport')
const Readline = require('readline')

const cors = require('cors')
const path = require('path')

const PORT = 3000
const hostname = 'localhost'

const BAUD_RATE = 9600
const SERIAL_PORT = 'COM3'

app.use(express.urlencoded({ extended: true }))
app.use(express.json())
app.use(cors())

const arduino = new SerialPort({
    path: SERIAL_PORT,
    baudRate: BAUD_RATE
})

const parserDados = arduino.pipe(new ReadlineParser({ delimiter: '\r\n' }))

arduino.on("open", () => {
    console.log(`Conexão com Arduino em ${SERIAL_PORT}`)
})

arduino.on("error", () => {
    console.error(`Erro na comunicação`, err)
})

parserDados.on('dados', (dados) => {
    const linha = dados.toString().trim()
    if (linha > 0) console.log(`Arduino: ${linha}`)
})

const teclado = Readline.createInterface({
    input: process.stdin,
    output: process.stdout
})

teclado.on('line', (input) => {
    const comando2 = input.trim()

    if (comando2 === 's' || comando2 === 'd') {
        arduino.write(comando2, (err) => {
            if (err) console.error('Erro ao enviar comando', err)
            else console.log(`Comando ${comando2} enviado`)
        })
    } else {
        console.log('Comando inválido.')
    }
})

app.get('/ligar', (req, res) => {
    arduino.write('s', (err) => {
        if (err) return res.status(500).json({ led: "erro" })
    })
})

app.listen(PORT, hostname, () => {
    console.log(`Servidor rodando em http://${hostname}:${PORT}`)
})

Existem vários modelos, como Arduino Uno, Mega e Nano.

Exemplo de uso: controlar LEDs, sensores, motores e IoT.

Por ser uma plataforma colaborativa, milhares de tutoriais e códigos estão disponíveis.

Entrada e Saída de Dados

Exemplificação | Funcionalidade | Utilização

Entrada e saída de dados (Input/Output - I/O) são processos essenciais na interação entre computadores e usuários ou dispositivos.


Entrada de Dados

A entrada ocorre quando o computador recebe informações para processar.


    Mouse Teclado Câmera Microfone Sensores

Saída de Dados

A saída ocorre quando o computador devolve informações processadas.


    Monitor Impressora Caixas de som LEDs Displays
Exemplo prático: Digitar no teclado (entrada) e ver o texto no monitor (saída).

Entrada e saída de dados são fundamentais para qualquer sistema computacional.

Script para Cálculos

Exemplificação | Funcionalidade | Utilização

Scripts para cálculos são usados para automatizar operações matemáticas, comuns em TI e eletrônica.

Linguagens como JavaScript, Python e C++ podem realizar cálculos simples ou complexos.


Como funcionam?

Um script lê valores (entrada), processa e exibe o resultado (saída).

Exemplo em JavaScript: 

let n1 = 10
let n2 = 5
let soma = n1 + n2
console.log("A soma é: " + soma)

Aplicações comuns

    Sistemas escolares
    Conversores de moeda
    Sensores IoT
    Dashboards e planilhas
    Simulações físicas
Exemplo com entrada: 

let num1 = prompt("Digite o primeiro número:")
let num2 = prompt("Digite o segundo número:")
let resultado = Number(num1) * Number(num2)
console.log("Resultado: " + resultado)

Scripts são essenciais em automação, cálculos e análise de dados.

Diodo Led & Display

Exemplificação | Funcionalidade | Utilização

O LED é um diodo emissor de luz muito usado em eletrônica e automação.

Em projetos com Arduino, representa uma saída física do sistema.


Relação com Scripts

Assim como scripts matemáticos, o código controla quando um LED acende ou apaga.

Exemplo simples em Arduino: 

void setup() {
    pinMode(13, OUTPUT);
}

void loop() {
    digitalWrite(13, HIGH);
    delay(1000);
    digitalWrite(13, LOW);
    delay(1000);
}

O LED é um indicador visual do funcionamento do código.


Aplicações e Importância

    Sinalização
    Testes I/O
    Monitoramento de sensores
    Ensino de eletrônica
    Baixo consumo de energia

O LED é uma forma simples e visual de comunicação entre hardware e programador.
