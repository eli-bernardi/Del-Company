O que é um SGBD? Tipos e Aplicações

Definição | Classificação | Utilização

O SGBD (Sistema Gerenciador de Banco de Dados) é um software responsável por armazenar, organizar e manipular dados de forma estruturada. Ele garante segurança, consistência e acesso controlado às informações.

Tipos de SGBD:

  • Relacional (RDBMS): dados em tabelas – MySQL, PostgreSQL.

  • Não Relacional (NoSQL): dados flexíveis – MongoDB, Redis.
  • Hierárquico: estrutura em árvore – IMS.
  • Orientado a objetos: armazena dados como objetos – ObjectDB.

Aplicações: sistemas corporativos, e-commerces, CRMs, redes sociais e softwares que tratam grandes volumes de dados.

Dicionário de Dados e Tipos

Definição | Estrutura | Tipos de Dados

O dicionário de dados documenta todas as informações de um banco de dados: nomes de tabelas, colunas, tipos e restrições. É essencial para manter a padronização e clareza do sistema.

Modelagem de Banco de Dados | DER e MER

Conceito | Estrutura | Diferença

A modelagem de banco de dados é o processo de planejar como os dados serão organizados e relacionados. Ela garante integridade, consistência e fácil manutenção.

DER (Diagrama Entidade-Relacionamento): representa graficamente as entidades, atributos e relacionamentos.

MER (Modelo Entidade-Relacionamento): é a parte conceitual, que define logicamente as relações entre os dados antes da implementação.

Exemplo de Diagrama DER

Visualização | Estrutura | Ligação

O diagrama mostra entidades como Aluno, Curso e Matrícula, conectadas por relacionamentos. Assim, o banco entende como os dados se relacionam entre si.

Exemplo de Diagrama MER

Representação | Entidades | Relacionamentos

O MER ilustra as relações de forma conceitual, sem se preocupar com implementações. Ele define entidades, atributos e cardinalidades, como “um aluno pode estar em vários cursos”.
